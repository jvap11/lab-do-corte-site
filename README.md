# Laboratório do Corte

Site institucional estático da barbearia Laboratório do Corte.

## Arquivos principais

- `index.html`: estrutura e conteúdo do site.
- `estilos.css`: visual, responsividade, cores e tipografia.
- `animacoes.js`: menu mobile, animações, carrossel e links do WhatsApp.
- `imagens/`: logo, favicon e fotos da galeria.
- `INSTRUCOES.md`: guia simples para editar textos, imagens, links e publicar.

## Como abrir localmente

Basta abrir o arquivo `index.html` no navegador.

## Deploy na Vercel

1. Suba este projeto para um repositório no GitHub.
2. Acesse https://vercel.com/.
3. Clique em `Add New Project`.
4. Escolha o repositório do GitHub.
5. Mantenha as configurações padrão.
6. Clique em `Deploy`.

Como é um site estático, não precisa instalar dependências nem rodar build.
